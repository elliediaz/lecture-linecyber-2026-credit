## 전개 (19분)

### 섹션 1: 상관분석 (7분)

#### 상관계수 개념 [3분]

> **상관분석**은 두 변수가 함께 변하는 정도를 측정합니다.
>
> 핵심 지표가 **피어슨 상관계수(r)**입니다. -1에서 +1 사이 값을 가져요.
>
> +1에 가까우면 강한 양의 상관, 하나가 증가하면 다른 것도 증가합니다. -1에 가까우면 강한 음의 상관, 하나가 증가하면 다른 것은 감소합니다. 0에 가까우면 관계가 없습니다.
>
> 대략적인 기준으로, 절대값 0.7 이상이면 강한 상관, 0.3~0.7이면 중간, 0.3 미만이면 약한 상관이라고 봅니다.

#### Python 실습 [2분]

> *(코드 시연)*
>
> ```python
> import numpy as np
> r = np.corrcoef(온도, 불량률)[0, 1]
> ```
>
> pandas DataFrame이라면 `.corr()` 메서드로 상관행렬을 한 번에 볼 수 있습니다.

#### 주의사항: 상관 ≠ 인과 [2분]

> 중요한 주의사항입니다. **상관관계가 인과관계는 아닙니다.**
>
> 유명한 예로, 아이스크림 판매량과 익사 사고 수가 높은 상관을 보입니다. 하지만 아이스크림이 익사를 유발하는 건 아니죠. 숨겨진 변수인 "기온" 때문입니다.
>
> 상관분석은 관계의 존재만 알려줍니다. 원인-결과는 별도 분석이 필요해요.

### 섹션 2: 선형회귀 (8분)

#### 회귀분석 개념 [3분]

> **회귀분석**은 변수 간 관계를 수식으로 표현합니다.
>
> Y = β₀ + β₁X
>
> X가 독립변수(원인), Y가 종속변수(결과)입니다. β₀는 절편, β₁은 기울기예요.
>
> 회귀분석의 목적은 두 가지입니다. 첫째, X가 Y에 미치는 영향을 파악. 둘째, X 값으로 Y 값을 예측.

#### sklearn 실습 [3분]

> sklearn으로 회귀 모델을 만들어봅시다.
>
> *(코드 시연)*
>
> ```python
> from sklearn.linear_model import LinearRegression
>
> model = LinearRegression()
> model.fit(X, y)
>
> print(model.intercept_)  # 절편
> print(model.coef_)       # 기울기
>
> y_pred = model.predict(X_new)  # 예측
> ```

#### 모델 평가 [2분]

> 모델이 얼마나 좋은지 **R² (결정계수)**로 평가합니다.
>
> R²는 0~1 사이 값이고, 1에 가까울수록 모델의 설명력이 좋습니다. 0.7 이상이면 보통 괜찮다고 봅니다.
>
> ```python
> r2 = model.score(X, y)
> ```
