#### 왜 시각화가 중요한가? [1분]

#### 기술통계량 [2분]

> **기술통계**는 데이터를 요약하는 숫자들입니다.
>
> **평균**은 모든 값을 더해서 개수로 나눈 것. **중앙값**은 정렬했을 때 가운데 값입니다.
>
> 그럼 둘 중 뭘 써야 할까요?
>
> 데이터가 [100, 100, 100, 100, 100, 100, 100, 100, 100, 1000]이라면?
> 평균은 190, 중앙값은 100입니다. 이상치(극단값)가 있으면 평균은 왜곡돼요.
>
> 그래서 **제조 데이터처럼 이상치가 많은 경우에는 중앙값도 함께** 확인해야 합니다.

#### 표준편차의 의미 [2분]

> **표준편차**는 데이터가 얼마나 퍼져있는지 보여줍니다.
>
> 라인 A 생산량: [1190, 1200, 1195, 1205, 1210] → 표준편차: 7.1
> 라인 B 생산량: [1000, 1100, 1200, 1300, 1400] → 표준편차: 141.4
>
> 두 라인 모두 평균은 1200입니다. 하지만 라인 A는 일관되고, 라인 B는 불안정해요.
>
> 표준편차가 작을수록 **품질이 일관**됩니다. 제조에서 매우 중요한 지표예요.

#### Matplotlib 소개 [2분]

> 이제 **Matplotlib**을 배워봅시다. Python의 표준 시각화 라이브러리입니다.
>
> ```python
> import matplotlib.pyplot as plt
>
> # 한글 폰트 설정 (Windows)
> plt.rcParams['font.family'] = 'Malgun Gothic'
> plt.rcParams['axes.unicode_minus'] = False
> ```
>
> 한글을 쓰려면 폰트 설정이 필요합니다. 이 설정 안 하면 한글이 네모로 깨져요.

#### 그래프 종류 선택 [1분]

> 어떤 그래프를 언제 쓸지 정리해드릴게요.
>
> - **히스토그램**: 데이터 분포를 볼 때

> - **산점도**: 두 변수 관계를 볼 때
> - **선 그래프**: 시간에 따른 변화를 볼 때
>
> 목적에 맞는 그래프를 선택하는 게 중요합니다.

## 실습편 (15-20분)

### 실습 소개 [2분]

> 이제 실습 시간입니다. 오늘은 제조 데이터를 시각화해보겠습니다.
>
> **실습 환경**을 확인해주세요.
>
> ```python
> import numpy as np
> import pandas as pd
> import matplotlib.pyplot as plt
>
> plt.rcParams['font.family'] = 'Malgun Gothic'
> plt.rcParams['axes.unicode_minus'] = False
> ```
>
> 모두 준비되셨으면 시작하겠습니다.

### 실습 1: 데이터 준비 [2분]

> 첫 번째 실습입니다. 샘플 데이터를 생성해봅시다.
>
> ```python
> np.random.seed(42)
> n = 100
>
> df = pd.DataFrame({
>     '생산량': np.random.normal(1200, 50, n),
>     '불량수': np.random.poisson(30, n),
>     '온도': np.random.normal(85, 5, n),
>     '라인': np.random.choice([1, 2, 3], n)
> })
>
> df['불량률'] = df['불량수'] / df['생산량']
> print(df.describe())
> ```
>
> 100개의 샘플 데이터가 생성되었습니다. describe()로 기본 통계를 확인하세요.

### 실습 2: 히스토그램 [3분]

> 두 번째 실습입니다. 생산량 분포를 히스토그램으로 그려봅시다.
>
> ```python
> plt.figure(figsize=(10, 6))
> plt.hist(df['생산량'], bins=20, edgecolor='black', alpha=0.7)
>
> mean_val = df['생산량'].mean()
> plt.axvline(mean_val, color='red', linestyle='--',
>             label=f'평균: {mean_val:.0f}')
>
> plt.xlabel('생산량')
> plt.ylabel('빈도')
> plt.title('생산량 분포')
> plt.legend()
> plt.show()
> ```
>
> bins는 막대 개수입니다. 20~30 정도가 적당해요.
> axvline으로 평균선을 추가했습니다. 분포의 중심이 어디인지 확인할 수 있죠.

### 실습 3: 상자그림 [3분]

> 세 번째 실습입니다. 라인별 생산량을 상자그림으로 비교해봅시다.
>
> ```python
> line1 = df[df['라인'] == 1]['생산량']
> line2 = df[df['라인'] == 2]['생산량']
> line3 = df[df['라인'] == 3]['생산량']
>
> plt.figure(figsize=(8, 6))
> plt.boxplot([line1, line2, line3],
>             labels=['라인1', '라인2', '라인3'])
> plt.ylabel('생산량')
> plt.title('라인별 생산량 분포')
> plt.show()
> ```
>
> 상자 바깥의 점(o)은 이상치입니다. 어떤 라인에 이상치가 많은지 확인해보세요.
>
> 상자의 길이가 길수록 데이터가 퍼져있다는 뜻입니다.

### 실습 4: 산점도 [3분]

> 네 번째 실습입니다. 온도와 불량률의 관계를 산점도로 확인해봅시다.
>
> ```python
> plt.figure(figsize=(10, 6))
> plt.scatter(df['온도'], df['불량률'], alpha=0.5, c='steelblue')
>
> plt.xlabel('온도 (도)')
> plt.ylabel('불량률')
> plt.title('온도와 불량률의 관계')
> plt.show()
> ```
>
> 점들이 오른쪽 위로 향하면 양의 상관관계 (온도가 높을수록 불량률도 높음)입니다.
> 점들이 무작위로 퍼져있으면 두 변수는 관련이 없는 거예요.

### 실습 5: 여러 그래프 한 번에 [3분]

> 다섯 번째 실습입니다. 4개의 그래프를 한 번에 그려봅시다.
>
> ```python
> fig, axes = plt.subplots(2, 2, figsize=(12, 10))
>
> axes[0, 0].hist(df['생산량'], bins=20, edgecolor='black')
> axes[0, 0].set_title('생산량 분포')
>
> axes[0, 1].boxplot([line1, line2, line3])
> axes[0, 1].set_title('라인별 생산량')
>
> axes[1, 0].scatter(df['온도'], df['불량률'], alpha=0.5)
> axes[1, 0].set_title('온도 vs 불량률')
>
> line_avg = df.groupby('라인')['생산량'].mean()
> axes[1, 1].bar(line_avg.index, line_avg.values)
> axes[1, 1].set_title('라인별 평균 생산량')
>
> plt.tight_layout()
> plt.show()
> ```
>
> subplots(2, 2)는 2행 2열, 총 4개 그래프를 만듭니다. tight_layout()은 레이아웃을 자동 조절해줍니다.

### 실습 6: 그래프 저장 [2분]

> 마지막 실습입니다. 그래프를 파일로 저장해봅시다.
>
> ```python
> plt.figure(figsize=(10, 6))
> plt.hist(df['생산량'], bins=20, edgecolor='black')
> plt.title('생산량 분포')
>
> plt.savefig('production_hist.png', dpi=300, bbox_inches='tight')
> print("그래프가 저장되었습니다.")
> plt.show()
> ```
>
> dpi=300은 고해상도입니다. 보고서나 문서에 넣을 때 권장해요.
