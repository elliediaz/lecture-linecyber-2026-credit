# AI 윤리 4대 원칙

## 1. 공정성 (Fairness)

- AI 시스템은 **모든 사용자를 공평하게** 대우해야 함
- 특정 그룹에 불리한 결과를 내면 안 됨

### 제조 현장 예시

```
품질 검사 AI가 A라인 제품만 더 엄격하게 판정한다면?
→ A라인 작업자에게 부당한 불이익
→ 학습 데이터의 편향이 원인일 수 있음
```

---

# AI 윤리 4대 원칙

## 2. 투명성 (Transparency)

- AI의 **의사결정 과정이 설명 가능**해야 함
- "왜 이런 결과가 나왔는지" 알 수 있어야 함

### 제조 현장 예시

```
불량 판정 AI: "이 제품은 불량입니다"
→ 작업자: "왜요? 어디가 문제인가요?"

좋은 AI: "표면 스크래치가 0.5mm 이상 검출되어 불량입니다"
```

---

# AI 윤리 4대 원칙

## 3. 책임성 (Accountability)

- AI 시스템의 결과에 대한 **책임 주체가 명확**해야 함
- 문제 발생 시 대응 체계 필요

### 질문: AI가 잘못 판단해서 불량품이 출하되었다면?

| 후보 | 책임 여부 |
|------|----------|
| AI 개발사 | 모델 오류 시 |
| 도입 기업 | 운영 관리 책임 |
| 담당자 | 검증 미흡 시 |

---

# AI 윤리 4대 원칙

## 4. 안전성 / 프라이버시

- 개인정보 및 기업 비밀 **보호 의무**
- 데이터 수집/저장/활용 시 동의 필요

### 제조 현장 예시

```
작업자 동작 분석 AI
→ 개인 행동 패턴 수집
→ 동의 없이 인사 평가에 활용?
→ 심각한 프라이버시 침해!
```
