## 전개 (19분)

### 섹션 1: 결측치 처리 (9분)

#### 결측치 탐지 [3분]

> **결측치**는 비어있는 값입니다. 센서 오류, 입력 누락, 통신 두절 등 다양한 이유로 발생해요.
>
> pandas에서는 isnull() 메서드로 탐지합니다.
>
> *(코드 시연)*
>
> ```python
> df.isnull().sum()  # 열별 결측치 수
> df.isnull().mean() * 100  # 결측치 비율
> ```

#### 결측치 처리 방법 [6분]

> 처리 방법은 크게 세 가지입니다.
>
> **첫째, 삭제**입니다. 결측치가 매우 적을 때 (5% 미만) 사용합니다.
>
> ```python
> df.dropna()  # 결측치 있는 행 삭제
> ```
>
> **둘째, 대체**입니다. 평균, 중앙값, 최빈값 등으로 채웁니다.
>
> ```python
> df['온도'].fillna(df['온도'].median())  # 중앙값 대체
> ```
>
> 이상치가 있으면 평균보다 중앙값이 낫습니다.
>
> **셋째, 보간**입니다. 시계열 데이터에서 앞뒤 값을 이용해 추정합니다.
>
> ```python
> df['온도'].interpolate()  # 선형 보간
> ```

### 섹션 2: 이상치 처리 (8분)

#### 이상치 탐지 [4분]

> **이상치**는 비정상적으로 극단적인 값입니다.
>
> 탐지 방법 두 가지를 배웁니다.
>
> **IQR 방법**: 사분위수 범위를 이용합니다.
>
> ```python
> Q1, Q3 = df['값'].quantile([0.25, 0.75])
> IQR = Q3 - Q1
> outliers = (df['값'] < Q1 - 1.5*IQR) | (df['값'] > Q3 + 1.5*IQR)
> ```
>
> **Z-score 방법**: 평균에서 표준편차 몇 배 떨어졌는지 봅니다.
>
> ```python
> z_scores = np.abs(stats.zscore(df['값']))
> outliers = z_scores > 3
> ```

#### 이상치 처리 [4분]

> 처리 옵션은 여러 가지입니다.
>
> 1. 삭제
> 2. 클리핑 (상/하한 설정)
> 3. 대체 (중앙값 등)
> 4. 별도 플래그 (is_outlier 열 추가)
>
> **중요**: 무조건 제거하지 마세요! 이상치가 중요한 정보일 수 있습니다.
>
> 불량품 예측에서는 이상치가 바로 우리가 찾는 것이잖아요.
