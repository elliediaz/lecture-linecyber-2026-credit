### 핵심 내용 (8분)

#### 머신러닝이란? [1.5분]

> **머신러닝**은 기계 학습이라고도 합니다.
>
> 핵심은 **명시적으로 프로그래밍하지 않아도 데이터로부터 스스로 학습**한다는 겁니다.
>
> 예를 들어볼게요. 전통적 프로그래밍에서는 "온도가 90도 넘으면 불량"이라고 직접 규칙을 만들어요.
>
> 머신러닝에서는? 데이터를 보고 모델이 알아서 "아, 온도 87도 넘으면 불량이 많네"라고 학습합니다.
>
> 사람이 규칙을 다 만들 필요가 없어요. 데이터가 규칙을 알려줍니다.

#### 핵심 용어 [1분]

> 머신러닝에서 꼭 알아야 할 용어가 있어요.
>
> **특성(Feature)**은 입력 데이터입니다. 온도, 습도, 속도 같은 거요. 보통 X로 표시합니다.
>
> **타겟(Target)**은 예측하려는 값입니다. 불량 여부, 생산량 같은 거요. y로 표시합니다.
>
> **모델(Model)**은 학습된 패턴입니다. X를 넣으면 y를 예측해주죠.

#### 머신러닝의 종류 [1분]

> 머신러닝은 크게 세 가지로 나눕니다.
>
> **지도학습**: 정답이 있는 데이터로 학습합니다. 우리가 집중할 부분이에요.
>
> **비지도학습**: 정답 없이 패턴을 발견합니다. 군집화나 이상 탐지에 쓰여요.
>
> **강화학습**: 보상 기반으로 학습합니다. 게임 AI나 로봇에 쓰여요.
>
> 이 과정에서는 **지도학습**에 집중합니다. 제조 현장에서 가장 많이 쓰이거든요.

#### 지도학습: 분류와 회귀 [2분]

> 지도학습에는 두 가지 종류가 있어요. **분류**와 **회귀**입니다.
>
> **분류(Classification)**는 범주를 예측합니다.
> "이 제품이 불량인가요?" → "네" 또는 "아니오"
> 출력이 정해진 카테고리 중 하나예요.
>
> **회귀(Regression)**는 숫자를 예측합니다.
> "생산량이 얼마나 될까요?" → "1,247개"
> 출력이 연속적인 숫자예요.
>
> 구분하는 쉬운 팁이 있어요.
> "~인가요?"라고 물으면 분류, "얼마나?"라고 물으면 회귀입니다.

#### sklearn 소개 [1.5분]

> 머신러닝을 쉽게 하려면 **sklearn**을 씁니다. scikit-learn이라고도 해요.
>
> sklearn의 장점은 모든 모델이 같은 패턴으로 동작한다는 겁니다.
>
> ```python
> model.fit(X, y)      # 학습
> model.predict(X)     # 예측
> model.score(X, y)    # 평가
> ```
>
> 의사결정트리든, 랜덤포레스트든, 신경망이든 다 이 패턴이에요.
> 한번 배우면 다른 모델도 같은 방식으로 쓸 수 있습니다.

#### 학습/테스트 분리 [1분]

> 중요한 개념이 하나 더 있어요. **학습 데이터와 테스트 데이터를 분리**해야 합니다.
>
> 왜일까요? 시험 문제를 미리 보면 안 되잖아요.
>
> 모델도 마찬가지입니다. 학습할 때 본 데이터로 평가하면 진짜 실력인지 알 수 없어요.
>
> 그래서 데이터를 80%는 학습용, 20%는 테스트용으로 나눕니다.
> 학습에 쓰지 않은 데이터로 성능을 평가해야 진짜 실력을 알 수 있습니다.

## 실습편 (15-20분)

### 실습 소개 [2분]

> 이제 실습 시간입니다. 오늘은 sklearn의 기본 사용법을 익힙니다.
>
> **실습 목표**입니다.
> 1. 분류 모델로 불량 여부를 예측합니다.
> 2. 회귀 모델로 생산량을 예측합니다.
> 3. sklearn의 fit, predict, score 패턴을 익힙니다.
>
> **실습 환경**을 확인해주세요.
>
> ```python
> import numpy as np
> import pandas as pd
> from sklearn.model_selection import train_test_split
> from sklearn.tree import DecisionTreeClassifier
> from sklearn.linear_model import LinearRegression
> ```

### 실습 1: 데이터 생성 [2분]

> 첫 번째 실습입니다. 제조 데이터를 생성합니다.
>
> ```python
> np.random.seed(42)
> n = 200
>
> df = pd.DataFrame({
>     '온도': np.random.normal(85, 5, n),
>     '습도': np.random.normal(50, 10, n),
>     '속도': np.random.normal(100, 15, n),
> })
> ```
>
> 온도, 습도, 속도 세 가지 특성이 있는 200개 데이터입니다.
>
> 불량 여부는 온도가 높을수록 확률이 높게 설정했어요.

### 실습 2: 특성과 타겟 분리 [2분]

> 두 번째 실습입니다. X와 y를 분리합니다.
>
> ```python
> # 분류 문제
> X_clf = df[['온도', '습도', '속도']]
> y_clf = df['불량여부']
>
> print(y_clf.value_counts())
> ```
>
> X는 특성들, y는 예측할 타겟입니다.
>
> 분류에서는 불량여부(0 또는 1), 회귀에서는 생산량(숫자)이 타겟이 됩니다.

### 실습 3: 학습/테스트 분리 [2분]

> 세 번째 실습입니다. 데이터를 학습용과 테스트용으로 나눕니다.
>
> ```python
> from sklearn.model_selection import train_test_split
>
> X_train, X_test, y_train, y_test = train_test_split(
>     X_clf, y_clf,
>     test_size=0.2,
>     random_state=42
> )
> ```
>
> 200개 중 160개는 학습용, 40개는 테스트용입니다.
>
> random_state를 지정하면 매번 같은 결과가 나와서 재현성이 보장됩니다.

### 실습 4: 분류 모델 [3분]

> 네 번째 실습입니다. 의사결정트리로 분류 모델을 만들어봅니다.
>
> ```python
> from sklearn.tree import DecisionTreeClassifier
>
> model = DecisionTreeClassifier(random_state=42)
> model.fit(X_train, y_train)
> ```
>
> 모델을 만들고 fit으로 학습시킵니다. 이게 끝이에요!
>
> 이제 예측해봅시다.
>
> ```python
> y_pred = model.predict(X_test)
> accuracy = (y_pred == y_test).mean()
> print(f"정확도: {accuracy:.1%}")
> ```
>
> 테스트 데이터로 예측하고 정확도를 계산합니다.

### 실습 5: 회귀 모델 [3분]

> 다섯 번째 실습입니다. 선형회귀로 회귀 모델을 만들어봅니다.
>
> ```python
> from sklearn.linear_model import LinearRegression
>
> X_train_r, X_test_r, y_train_r, y_test_r = train_test_split(
>     X_reg, y_reg, test_size=0.2, random_state=42
> )
>
> model_reg = LinearRegression()
> model_reg.fit(X_train_r, y_train_r)
> ```
>
> 패턴이 똑같죠? 분류든 회귀든 fit, predict, score입니다.
>
> ```python
> r2 = model_reg.score(X_test_r, y_test_r)
> print(f"R² 점수: {r2:.3f}")
> ```
>
> 회귀에서는 R² 점수로 성능을 평가합니다. 1에 가까울수록 좋아요.

### 실습 6: 문제 유형 구분 [2분]

> 여섯 번째 실습입니다. 분류와 회귀를 구분하는 연습을 해봅시다.
>
> "제품이 불량인가요?" → 분류 (예/아니오)
> "불량률이 몇 %?" → 회귀 (숫자)
> "품질 등급은?" → 분류 (A/B/C)
> "생산량은 얼마?" → 회귀 (숫자)
>
> "~인가요?"면 분류, "얼마나?"면 회귀. 이것만 기억하세요.
