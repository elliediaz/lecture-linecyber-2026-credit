## 전개 (19분)

### 섹션 1: 시계열 데이터 이해 (5분)

#### 시계열 데이터란 [2min]

> **시계열 데이터(Time Series Data)**는 시간 순서에 따라 관측된 데이터입니다.
>
> 제조 현장 예시로는:
> - 일별 생산량
> - 시간대별 불량률
> - 분 단위 센서 온도
>
> 비즈니스에서는:
> - 월별 매출액
> - 주별 방문자 수

#### 일반 데이터와 차이 [3min]

> 시계열 데이터는 일반 데이터와 다른 특성이 있어요.
>
> **첫째, 순서가 중요합니다.** 일반 데이터는 행 순서를 바꿔도 괜찮지만, 시계열은 시간 순서가 의미를 가집니다.
>
> **둘째, 시간 의존성이 있습니다.** 오늘 생산량은 어제 생산량과 관련이 있어요. 이것을 자기상관(Autocorrelation)이라고 합니다.
>
> **셋째, 계절성이 있을 수 있습니다.** 12월은 연말 특수로 매출이 높고, 여름에는 아이스크림 판매가 늘어나죠.

---

### 섹션 2: 날짜/시간 처리 (7분)

#### datetime 모듈 [2min]

> Python에서 날짜와 시간은 **datetime** 모듈로 다룹니다.
>
> *(코드 시연)*
>
> ```python
> from datetime import datetime, timedelta
>
> # 현재 시간
> now = datetime.now()
>
> # 특정 날짜 생성
> date = datetime(2024, 1, 15)
>
> # 문자열 → datetime
> date = datetime.strptime('2024-01-15', '%Y-%m-%d')
> ```

#### Pandas 날짜 처리 [3min]

> Pandas에서는 **pd.to_datetime**을 사용합니다. 더 편리해요.
>
> *(코드 시연)*
>
> ```python
> # 문자열 열을 datetime으로 변환
> df['날짜'] = pd.to_datetime(df['날짜'])
>
> # 날짜 정보 추출
> df['연도'] = df['날짜'].dt.year
> df['월'] = df['날짜'].dt.month
> df['요일'] = df['날짜'].dt.dayofweek  # 0=월요일
> ```
>
> dt 접근자를 사용해서 연도, 월, 일, 요일 등을 쉽게 추출할 수 있어요.

#### 날짜 인덱스 [2min]

> 시계열 분석에서는 **날짜를 인덱스로** 설정하는 것이 편리합니다.
>
> ```python
> df = df.set_index('날짜')
>
> # 이제 날짜로 필터링 가능
> df['2024-01']  # 2024년 1월 전체
> df['2024-01-01':'2024-06-30']  # 기간 지정
> ```

### 섹션 3: 시계열 시각화와 기법 (7min)

#### 기본 시각화 [2min]

> *(코드 시연)*
>
> ```python
> plt.figure(figsize=(12, 5))
> plt.plot(df.index, df['생산량'])
> plt.title('일별 생산량 추이')
> plt.xticks(rotation=45)
> plt.show()
> ```
>
> 시계열은 **선 그래프**로 시각화합니다. X축이 시간, Y축이 값이에요.

#### 리샘플링 [2min]

> **리샘플링(Resampling)**은 집계 주기를 바꾸는 것입니다.
>
> ```python
> # 일별 → 월별 평균
> monthly = df.resample('M').mean()
>
> # 일별 → 주별 합계
> weekly = df.resample('W').sum()
> ```
>
> 일별 데이터가 너무 세밀하면 월별이나 주별로 집계해서 보면 패턴이 더 잘 보여요.

#### 이동평균 [3min]

> **이동평균(Rolling Mean)**은 노이즈를 줄이고 추세를 파악하는 기법입니다.
>
> *(코드 시연)*
>
> ```python
> df['이동평균_7일'] = df['생산량'].rolling(window=7).mean()
> ```
>
> 7일 이동평균은 최근 7일의 평균이에요. 어제까지 7일, 오늘까지 7일... 이렇게 움직이면서 평균을 계산합니다.
>
> 일별 변동은 심하지만 이동평균을 그리면 전체적인 추세가 보여요. "아, 최근에 증가 추세구나" 같은 것을 파악할 수 있죠.
