### 섹션 1: 기술통계 이해하기 (5분)

#### 중심 경향 지표 [2분]

> 먼저 **기술통계**가 뭔지 알아봅시다. 데이터를 요약하는 숫자들이에요.
>
> 대표적인 게 **평균, 중앙값, 최빈값**입니다. 이걸 "중심 경향"이라고 해요. 데이터의 중심이 어디인지 알려주는 거죠.
>
> **평균**은 다 아시죠? 모든 값을 더해서 개수로 나눈 것.
>
> **중앙값**은 데이터를 정렬했을 때 가운데 있는 값입니다.
>
> 그럼 둘 중 뭘 써야 할까요?
>
> *(예시 제시)*
>
> 데이터가 [100, 100, 100, 100, 100, 100, 100, 100, 100, 1000]이라고 해봅시다.
> 평균은 190이에요. 하지만 중앙값은 100입니다.
>
> 이상치, 즉 극단값이 있으면 평균은 왜곡됩니다. 그래서 **제조 데이터처럼 이상치가 많은 경우에는 중앙값도 함께** 확인해야 해요.

#### 퍼짐 정도 지표 [2분]

> 다음은 데이터가 얼마나 **퍼져있는지** 보는 지표입니다.
>
> 가장 중요한 게 **표준편차**예요.
>
> *(예시 제시)*
>
> 라인 A 생산량: 1190, 1200, 1195, 1205, 1210
> 라인 B 생산량: 1000, 1100, 1200, 1300, 1400
>
> 두 라인 모두 평균은 1200입니다. 하지만 라인 A의 표준편차는 7, 라인 B는 141이에요.
>
> 표준편차가 작다는 건 **품질이 일관**되다는 뜻입니다. 제조에서는 이게 매우 중요해요.

#### 왜 시각화가 필요한가 [1분]

### 섹션 2: Matplotlib 기초 (6분)

#### Matplotlib 소개 및 설정 [2분]

> 이제 **Matplotlib**을 배워봅시다. Python의 표준 시각화 라이브러리입니다.
>
> *(코드 시연)*
>
> ```python
> import matplotlib.pyplot as plt
>
> # 한글 설정 (Windows)
> plt.rcParams['font.family'] = 'Malgun Gothic'
> plt.rcParams['axes.unicode_minus'] = False
> ```
>
> `matplotlib.pyplot`을 `plt`로 줄여서 부릅니다. 관례예요.
>
> 한글을 쓰려면 폰트 설정이 필요합니다. Windows에서는 '맑은 고딕'을 쓰면 됩니다. 이 설정 안 하면 한글이 네모로 깨져요.

#### 기본 그래프 그리기 [2분]

> 가장 기본적인 선 그래프를 그려봅시다.
>
> *(코드 시연)*
>
> ```python
> x = [1, 2, 3, 4, 5]
> y = [2, 4, 6, 8, 10]
>
> plt.plot(x, y)
> plt.xlabel("날짜")
> plt.ylabel("생산량")
> plt.title("일별 생산량")
> plt.show()
> ```
>
> `plot()`으로 그래프를 그리고, `xlabel`, `ylabel`, `title`로 레이블을 붙입니다.
>
> 마지막에 `show()`를 호출해야 화면에 표시됩니다. Jupyter에서는 자동으로 나올 수도 있지만, 명시적으로 호출하는 습관을 들이세요.

#### 그래프 종류 선택 [2분]

> 어떤 그래프를 언제 쓸지 정리해드릴게요.
>
> **히스토그램**: 데이터 분포를 볼 때
> **상자그림**: 이상치와 범위를 볼 때
> **산점도**: 두 변수 관계를 볼 때
> **선 그래프**: 시간에 따른 변화를 볼 때
> **막대 그래프**: 범주별 비교할 때
>
> 목적에 맞는 그래프를 선택하는 게 중요합니다.

### 섹션 3: 주요 그래프 실습 (8분)

#### 히스토그램 [2.5분]

> 먼저 **히스토그램**입니다. 데이터가 어떻게 분포되어 있는지 보여주는 그래프예요.
>
> *(코드 시연)*
>
> ```python
> import numpy as np
>
> # 생산량 데이터 (평균 1200, 표준편차 50)
> data = np.random.normal(1200, 50, 1000)
>
> plt.figure(figsize=(10, 6))
> plt.hist(data, bins=30, edgecolor='black', alpha=0.7)
> plt.xlabel('생산량')
> plt.ylabel('빈도')
> plt.title('생산량 분포')
> plt.axvline(np.mean(data), color='red', linestyle='--', label='평균')
> plt.legend()
> plt.show()
> ```
>
> `bins`는 막대 개수입니다. 너무 적으면 분포가 안 보이고, 너무 많으면 노이즈가 보여요. 보통 20~30 정도가 적당합니다.
>
> 정규분포면 종 모양이 나오고, 한쪽으로 치우치면 꼬리가 긴 모양이 됩니다.

#### 상자그림 [2.5분]

> **상자그림(Box Plot)**은 5가지 통계를 한눈에 보여줍니다.
>
> *(상자그림 구조 설명)*
>
> 상자의 아래가 Q1(25%), 중간선이 중앙값(50%), 위가 Q3(75%)입니다. 수염이 최소/최대값이고, 그 바깥의 점들이 이상치예요.
>
> *(코드 시연)*
>
> ```python
> line1 = np.random.normal(1200, 30, 100)
> line2 = np.random.normal(1180, 50, 100)
> line3 = np.random.normal(1220, 40, 100)
>
> plt.figure(figsize=(8, 6))
> plt.boxplot([line1, line2, line3], labels=['라인1', '라인2', '라인3'])
> plt.ylabel('생산량')
> plt.title('라인별 생산량 분포')
> plt.show()
> ```
>
> 상자그림으로 보면 어떤 라인이 안정적인지, 이상치가 많은 라인은 어딘지 한눈에 파악할 수 있습니다.

#### 산점도 [2분]

> **산점도(Scatter Plot)**는 두 변수 사이의 관계를 보여줍니다.
>
> *(코드 시연)*
>
> ```python
> temp = np.random.normal(85, 5, 100)
> defect_rate = 0.02 + 0.005 * (temp - 80) + np.random.normal(0, 0.01, 100)
>
> plt.figure(figsize=(10, 6))
> plt.scatter(temp, defect_rate, alpha=0.5)
> plt.xlabel('온도 (°C)')
> plt.ylabel('불량률')
> plt.title('온도와 불량률의 관계')
> plt.show()
> ```
>
> 점들이 오른쪽 위로 향하면 **양의 상관관계** (온도가 높을수록 불량률도 높음), 왼쪽 아래로 향하면 **음의 상관관계**입니다.
>
> 점들이 무작위로 퍼져있으면 두 변수는 관련이 없는 거예요.

#### 여러 그래프 한 번에 [1분]

> 분석할 때는 여러 그래프를 한 번에 보는 게 좋습니다. `subplot`을 사용합니다.
>
> *(코드 시연)*
>
> ```python
> fig, axes = plt.subplots(2, 2, figsize=(12, 10))
>
> axes[0, 0].hist(data, bins=20)
> axes[0, 0].set_title('분포')
>
> axes[0, 1].boxplot(data)
> axes[0, 1].set_title('상자그림')
>
> # ... 나머지 그래프
>
> plt.tight_layout()
> plt.show()
> ```
>
> `subplots(2, 2)`는 2행 2열, 총 4개 그래프를 만듭니다.
