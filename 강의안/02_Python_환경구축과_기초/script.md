### 섹션 2: Python 기본 자료형 (7분)

#### 자료형 개요 [1분]

> 이제 Python 문법을 배워봅시다. 먼저 **자료형**입니다.
>
> 자료형이란 데이터의 종류를 말합니다. 숫자, 문자, 목록 등 데이터 종류에 따라 다르게 처리해야 하기 때문에 구분하는 거예요.
>
> Python에서 꼭 알아야 할 자료형은 5가지입니다. 정수, 실수, 문자열, 리스트, 딕셔너리.

#### 정수와 실수 [1.5분]

> 먼저 **정수와 실수**입니다.
>
> 정수는 int, 소수점 없는 숫자입니다. 생산량 1000개, 라인 번호 3번 같은 거죠.
>
> 실수는 float, 소수점 있는 숫자입니다. 불량률 0.025, 온도 85.7도 같은 것들이요.
>
> *(코드 시연)*
>
> ```python
> count = 1000
> rate = 0.025
> print(count + 100)  # 1100
> print(rate * 100)   # 2.5
> ```
>
> 사칙연산은 우리가 아는 그대로 됩니다. 더하기, 빼기, 곱하기, 나누기. 추가로 `**`는 거듭제곱, `%`는 나머지를 구합니다.

#### 문자열 [1.5분]

> 다음은 **문자열**, str입니다.
>
> 따옴표로 감싼 텍스트가 문자열이에요. 작은따옴표든 큰따옴표든 상관없습니다.
>
> *(코드 시연)*
>
> ```python
> product = "센서 모듈 A"
> line = 3
> message = f"제품: {product}, 라인: {line}번"
> print(message)  # 제품: 센서 모듈 A, 라인: 3번
> ```
>
> 여기서 중요한 게 **f-string**입니다. 문자열 앞에 f를 붙이고 중괄호 안에 변수를 넣으면 값이 자동으로 들어갑니다. 이거 정말 많이 쓰니까 꼭 기억해두세요.

#### 리스트 [1.5분]

> **리스트**는 여러 데이터를 순서대로 담는 자료형입니다. 대괄호로 만들어요.
>
> *(코드 시연)*
>
> ```python
> production = [1200, 1150, 1300, 1180, 1250]
> print(production[0])   # 1200 (첫 번째)
> print(production[-1])  # 1250 (마지막)
> ```
>
> 중요한 점! **인덱스는 0부터 시작**합니다. 첫 번째 요소가 0번, 두 번째가 1번입니다. 프로그래밍에서는 다 이렇게 세요.
>
> -1은 마지막 요소를 의미합니다. -2는 뒤에서 두 번째고요.

#### 딕셔너리 [1.5분]

> 마지막으로 **딕셔너리**입니다. 중괄호로 만들고, 키와 값의 쌍으로 데이터를 저장합니다.
>
> *(코드 시연)*
>
> ```python
> sensor = {
>     "온도": 85.2,
>     "습도": 45,
>     "상태": "정상"
> }
> print(sensor["온도"])  # 85.2
> ```
>
> 리스트는 순서대로 0, 1, 2번으로 접근하는데, 딕셔너리는 이름표인 "키"로 접근합니다.
>
> 실무에서 센서 데이터, 설비 정보 같은 걸 다룰 때 딕셔너리를 많이 씁니다.

### 섹션 3: 조건문과 반복문 (6분)

#### 조건문 if [2분]

> 이제 **조건문**을 배워봅시다. "만약 ~라면 ~해라"라는 논리를 코드로 표현하는 거예요.
>
> *(코드 시연)*
>
> ```python
> defect_rate = 0.03
>
> if defect_rate > 0.05:
>     print("경고: 불량률이 높습니다!")
> elif defect_rate > 0.02:
>     print("주의: 모니터링이 필요합니다.")
> else:
>     print("정상: 품질 기준 충족")
> ```
>
> if 다음에 조건을 쓰고 콜론을 찍습니다. 그 다음 줄은 **들여쓰기**를 해야 해요. 이 들여쓰기가 Python에서는 매우 중요합니다.
>
> elif는 "else if"의 줄임말로, 앞 조건이 거짓일 때 다음 조건을 확인합니다. else는 모든 조건이 거짓일 때 실행됩니다.

#### 비교 연산자와 논리 연산자 [1.5분]

> 조건을 만들 때 사용하는 **비교 연산자**들이 있습니다.
>
> `==`는 같다, `!=`는 다르다, `>`, `<`, `>=`, `<=`는 크기 비교입니다.
>
> 여기서 주의할 점! 같다를 비교할 때 등호를 **두 개** 씁니다. 등호 하나는 "할당"이에요.
>
> 여러 조건을 조합할 때는 **논리 연산자**를 씁니다.
>
> ```python
> if temp > 80 and humidity > 50:
>     print("온습도 모두 높음")
> ```
>
> `and`는 둘 다 참일 때, `or`는 하나라도 참일 때, `not`은 조건을 반전시킵니다.

#### 반복문 for [2.5분]

> **반복문**은 같은 작업을 여러 번 실행할 때 사용합니다.
>
> *(코드 시연)*
>
> ```python
> temperatures = [82, 85, 88, 95, 84]
>
> for temp in temperatures:
>     if temp > 90:
>         print(f"[경고] {temp}°C")
>     else:
>         print(f"[정상] {temp}°C")
> ```
>
> for 다음에 변수 이름을 쓰고, in 다음에 반복할 대상을 씁니다. 리스트의 각 요소가 차례로 temp 변수에 들어가면서 반복됩니다.
>
> `range()` 함수도 많이 씁니다.
>
> ```python
> for i in range(5):
>     print(f"라인 {i+1} 점검")
> # 라인 1 점검, 라인 2 점검, ... 라인 5 점검
> ```
>
> `range(5)`는 0부터 4까지 5개의 숫자를 만들어줍니다.
