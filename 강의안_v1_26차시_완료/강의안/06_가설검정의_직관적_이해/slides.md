# 왜 가설검정이 필요한가?

## 제조 현장의 질문들

- "새 공정이 불량률을 줄였을까?"
- "라인 A와 라인 B의 생산성 차이가 있을까?"
- "새 설비가 정말 효과가 있을까?"

> 이런 질문에 **객관적으로** 답하려면 통계적 검정이 필요

---

# 가설검정의 기본 구조

## 두 가지 가설

```
귀무가설 (H₀): 차이가 없다 / 효과가 없다
  → "새 공정은 불량률에 영향을 주지 않는다"

대립가설 (H₁): 차이가 있다 / 효과가 있다
  → "새 공정은 불량률을 줄인다"
```

### 접근 방식
1. 귀무가설이 참이라고 가정
2. 관측된 데이터가 얼마나 "이상한지" 계산
3. 너무 이상하면 귀무가설 기각

---

# p-value란?

## 귀무가설이 참일 때 관측 결과가 나올 확률

```
p-value = P(관측된 결과 이상으로 극단적 | H₀ 참)
```

### 해석
- **p-value가 작다** → 귀무가설 하에서 일어나기 어려운 결과 → H₀ 기각
- **p-value가 크다** → 귀무가설로 충분히 설명 가능 → H₀ 유지

### 일반적 기준
- p < 0.05: 통계적으로 유의미 (H₀ 기각)
- p < 0.01: 매우 유의미

---

# p-value 직관적 이해

## 동전 던지기 예시

```
동전을 10번 던져 앞면이 9번 나왔다.
이 동전이 공정한가?

H₀: 동전은 공정하다 (p = 0.5)
H₁: 동전은 공정하지 않다

p-value = P(9번 이상 앞면 | 공정한 동전)
        = 0.0107 (약 1%)

→ p < 0.05이므로 "공정하지 않다"고 결론
```

# t-검정 (t-test)

## 두 그룹의 평균 비교

```python
from scipy import stats

# 라인A와 라인B의 생산량
line_a = [1200, 1180, 1210, 1195, 1205]
line_b = [1150, 1140, 1160, 1155, 1145]

# 독립표본 t-검정
t_stat, p_value = stats.ttest_ind(line_a, line_b)

print(f"t-통계량: {t_stat:.3f}")
print(f"p-value: {p_value:.4f}")

if p_value < 0.05:
    print("두 라인의 생산량 차이가 유의미함")
```

# A/B 테스트

## 두 가지 버전 비교

### 개념
- A: 기존 방식 (Control)
- B: 새로운 방식 (Treatment)
- 무작위 배정 후 결과 비교

### 제조 현장 예시
```
A그룹: 기존 공정 → 불량률 3.2%
B그룹: 새 공정 → 불량률 2.8%

차이가 통계적으로 유의미한가?
```

---

# A/B 테스트 실습

## Python 코드

```python
from scipy import stats

# 불량품 수
a_defects = 32  # 1000개 중 32개
b_defects = 28  # 1000개 중 28개
n = 1000

# 비율 검정
count = np.array([a_defects, b_defects])
nobs = np.array([n, n])

stat, p_value = stats.proportions_ztest(count, nobs)

print(f"p-value: {p_value:.4f}")
```

# 검정력과 표본 크기

## 잘못된 결론의 위험

### 1종 오류 (α)
- 실제로 효과가 없는데 있다고 결론
- 일반적으로 5%로 설정

### 2종 오류 (β)
- 실제로 효과가 있는데 없다고 결론
- 검정력 = 1 - β

### 표본 크기
- 표본이 작으면 진짜 차이를 놓칠 수 있음
- 충분한 표본 크기 확보 필요

---

# 통계적 유의성 vs 실용적 유의성

## 주의할 점

```
표본이 매우 크면 아주 작은 차이도 "유의미"

예: n = 100,000
라인A 불량률: 3.00%
라인B 불량률: 2.99%
p-value = 0.03 (유의미!)

→ 통계적으로는 유의미하지만
→ 실제로 의미 있는 차이인가?
```

> **항상 효과 크기(effect size)도 함께 확인**

# 가설검정 절차 요약

## 5단계 프로세스

1. **가설 설정**: H₀와 H₁ 정의
2. **유의수준 결정**: 보통 α = 0.05
3. **데이터 수집**: 적절한 표본 크기
4. **검정 통계량 계산**: t-값, p-value
5. **결론 도출**: p < α면 H₀ 기각
