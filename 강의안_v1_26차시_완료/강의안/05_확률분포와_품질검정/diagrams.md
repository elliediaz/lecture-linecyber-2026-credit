# [5차시] 확률분포와 품질 검정 - 다이어그램

## 1. 확률분포가 필요한 이유

```mermaid
flowchart TD
    A[제조 현장의 질문] --> B["이 측정값이 정상인가?"]
    A --> C["두 라인의 차이가 있는가?"]
    A --> D["이 생산량이 이상치인가?"]

    B --> E[확률분포 이해]
    C --> E
    D --> E

    E --> F[데이터 기반 의사결정]
    F --> G[객관적 품질 관리]
```

## 2. 정규분포 구조

```mermaid
flowchart TD
    subgraph 정규분포["정규분포 (Normal Distribution)"]
        A["종 모양 (Bell Curve)"]
        B["평균(μ)을 중심으로 대칭"]
        C["평균에서 멀수록 확률 감소"]
    end

    D["생산량 데이터"] --> 정규분포
    E["측정 오차"] --> 정규분포
    F["품질 수치"] --> 정규분포

    정규분포 --> G["68-95-99.7 규칙 적용"]
```

## 3. 68-95-99.7 규칙

```mermaid
flowchart LR
    subgraph 규칙["68-95-99.7 규칙"]
        A["±1σ: 68%"]
        B["±2σ: 95%"]
        C["±3σ: 99.7%"]
    end

    D["μ=1200, σ=50"] --> E["±1σ: 1150~1250"]
    E --> F["68% 데이터"]

    D --> G["±2σ: 1100~1300"]
    G --> H["95% 데이터"]

    D --> I["±3σ: 1050~1350"]
    I --> J["99.7% 데이터"]

    J --> K["범위 밖 = 이상치"]
```

## 4. Z-score 계산

```mermaid
flowchart TD
    A["관측값: X"] --> B["Z-score 계산"]
    C["평균: μ"] --> B
    D["표준편차: σ"] --> B

    B --> E["Z = (X - μ) / σ"]

    E --> F{"|Z| 값은?"}

    F --> |"≤ 1"| G["정상 (68%)"]
    F --> |"1~2"| H["주의 (27%)"]
    F --> |"2~3"| I["경고 (4.3%)"]
    F --> |"> 3"| J["이상치 (0.3%)"]
```

## 5. Z-score 해석 기준

```mermaid
mindmap

      |Z| > 3

        즉시 조사
```

## 6. 가설검정 구조

```mermaid
flowchart TD
    A["질문: 새 공정이 효과 있는가?"] --> B["가설 설정"]

    B --> C["귀무가설 H₀<br>차이가 없다"]
    B --> D["대립가설 H₁<br>차이가 있다"]

    C --> E["데이터 수집"]
    D --> E

    E --> F["t-검정 수행"]

    F --> G["p-value 계산"]

    G --> H{p < 0.05?}

    H --> |예| I["H₀ 기각<br>차이가 있다!"]
    H --> |아니오| J["H₀ 유지<br>차이 없다고 볼 수 없음"]
```

## 7. p-value 해석

```mermaid
flowchart LR
    subgraph pvalue["p-value란?"]
        A["귀무가설이 참일 때<br>관측된 차이가<br>우연히 나타날 확률"]
    end

    pvalue --> B{판단 기준}

    B --> |"p < 0.01"| C["매우 유의미<br>강한 증거"]
    B --> |"p < 0.05"| D["유의미<br>차이 있음"]
    B --> |"p ≥ 0.05"| E["유의미하지 않음<br>차이 없다고 볼 수 없음"]

    C --> F["H₀ 기각"]
    D --> F
    E --> G["H₀ 유지"]
```

## 8. t-검정 과정

```mermaid
flowchart TD
    A["라인 A 데이터"] --> C["독립표본 t-검정"]
    B["라인 B 데이터"] --> C

    C --> D["t-통계량 계산"]
    D --> E["p-value 도출"]

    E --> F{p < 0.05?}

    F --> |예| G["두 라인 평균이<br>통계적으로 다름"]
    F --> |아니오| H["두 라인 평균 차이가<br>유의미하지 않음"]

    G --> I["라인 B 품질 개선 필요"]
```

## 9. 이상치 탐지 워크플로우

```mermaid
flowchart TD
    A[일별 생산 데이터] --> B[기술통계 계산]
    B --> C["평균, 표준편차"]

    C --> D[Z-score 계산]
    D --> E{|Z| > 2?}

    E --> |예| F[이상치 플래그]
    E --> |아니오| G[정상 데이터]

    F --> H[원인 조사]
    H --> I["설비 이상?<br>작업자 실수?<br>원자재 문제?"]

    I --> J[조치 및 기록]
```

## 10. 품질 검정 판단 기준

```mermaid
flowchart TD
    subgraph 개별값["개별 데이터 검정"]
        A[Z-score 계산] --> B{|Z| > 2?}
        B --> |예| C[이상치]
        B --> |아니오| D[정상]
    end

    subgraph 그룹비교["그룹 간 비교"]
        E[t-검정] --> F{p < 0.05?}
        F --> |예| G[유의미한 차이]
        F --> |아니오| H[차이 없음]
    end

    I[품질 데이터] --> 개별값
    I --> 그룹비교
```

## 11. 정규분포와 품질 관리 기준

```mermaid
flowchart LR
    subgraph 관리한계["품질 관리 한계"]
        A["UCL = μ + 3σ<br>상한 관리선"]
        B["CL = μ<br>중심선"]
        C["LCL = μ - 3σ<br>하한 관리선"]
    end

    D["측정값"] --> E{범위 내?}

    E --> |"LCL < 값 < UCL"| F[정상 운전]
    E --> |"값 > UCL 또는<br>값 < LCL"| G[이상 발생]

    G --> H[공정 조치 필요]
```
