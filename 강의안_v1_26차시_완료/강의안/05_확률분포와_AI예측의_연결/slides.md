# 확률이란?

## 불확실한 상황에서의 판단

- **빈도적 해석**: 반복 시행 시 발생 비율
- **베이지안 해석**: 믿음의 정도, 불확실성

### 제조 현장 예시
```
불량품 발생 확률 = 3%
→ 100개 중 평균 3개가 불량
→ 다음 제품이 불량일 가능성은 3%
```

---

# 확률분포란?

## 가능한 결과와 그 확률의 매핑

```
예: 품질 등급 분포

등급 A: 60%  ████████████
등급 B: 30%  ██████
등급 C: 10%  ██

합계: 100%
```

> 확률분포 = 모든 가능한 결과의 확률을 모은 것

---

# 정규분포 (Normal Distribution)

## 가장 중요한 확률분포

### 특성
- 평균을 중심으로 **대칭인 종 모양**
- 평균에서 멀어질수록 확률 감소
- **68-95-99.7 규칙** (경험적 법칙)

### 왜 중요한가?
- 자연현상의 대부분이 정규분포
- 제조 공정의 측정값도 정규분포
- 많은 AI 알고리즘이 정규분포 가정

---

# 68-95-99.7 규칙

## 표준편차와 확률의 관계

```
       ← 99.7% (±3σ) →
     ←── 95% (±2σ) ──→
      ←─ 68% (±1σ) ─→

          ▄▄▄
        ▄█████▄
      ▄█████████▄
   ▄▄███████████████▄▄
━━━━━━━━━━━━━━━━━━━━━━━━
  μ-3σ μ-2σ μ-σ  μ  μ+σ μ+2σ μ+3σ
```

---

# 정규분포 활용 예시

## 품질 관리에서의 활용

```python
# 생산량이 정규분포 N(1200, 50²)를 따를 때

평균(μ) = 1200
표준편차(σ) = 50

# 68%의 데이터가 1150~1250 범위에 존재
# 95%의 데이터가 1100~1300 범위에 존재
# 99.7%의 데이터가 1050~1350 범위에 존재

# 1050 미만 또는 1350 초과 → 이상치 의심!
```

---

# Python으로 정규분포

## scipy.stats 활용

```python
from scipy import stats
import numpy as np

# 정규분포 객체 생성
dist = stats.norm(loc=1200, scale=50)  # 평균=1200, 표준편차=50

# 확률 계산
prob = dist.cdf(1250)  # P(X ≤ 1250)
print(f"1250 이하 확률: {prob:.2%}")

# 랜덤 샘플 생성
samples = dist.rvs(size=1000)
```

# 이항분포 (Binomial Distribution)

## 성공/실패의 반복 시행

```python
# 불량률 3%인 제품 100개 검사 시
# 불량품 개수의 분포

from scipy import stats

n = 100  # 시행 횟수
p = 0.03  # 불량 확률

dist = stats.binom(n, p)

# 불량품이 정확히 3개일 확률
print(f"P(X=3): {dist.pmf(3):.2%}")

# 불량품이 5개 이하일 확률
print(f"P(X≤5): {dist.cdf(5):.2%}")
```

# AI 예측과 확률분포

## 예측값 + 불확실성

```
전통적 예측:
"내일 생산량은 1200개입니다"

확률적 예측:
"내일 생산량은 1200개로 예상되며,
 95% 신뢰구간은 [1100, 1300]입니다"
```

> AI 모델은 점 추정이 아닌 **분포**를 출력할 수 있음

---

# 확률적 예측의 장점

## 의사결정에 필요한 정보

1. **중심값**: 가장 가능성 높은 예측
2. **불확실성**: 예측의 신뢰도
3. **위험 평가**: 최악/최선의 시나리오

### 예시
```
불량률 예측: 3% (95% CI: 1.5% ~ 4.5%)

→ 최악의 경우 4.5%까지 가능
→ 추가 검사 자원 배치 결정에 활용
```

# 중심극한정리 (CLT)

## 왜 정규분포가 많이 나타나는가?

> 충분히 큰 표본의 평균은 **원래 분포와 상관없이**
> 정규분포에 가까워진다

### 의미
- 표본 크기 30 이상이면 대체로 적용
- 평균을 사용한 분석에 정규분포 가정 가능
- 많은 통계 기법의 이론적 근거

# 표준화 (Z-score)

## 다른 분포도 비교 가능하게

```python
# Z-score = (관측값 - 평균) / 표준편차

z = (observed - mean) / std

# 예: 생산량 1300개의 Z-score
# 평균 1200, 표준편차 50일 때
z = (1300 - 1200) / 50  # z = 2.0

# Z-score 해석
# |z| > 2: 상위/하위 5% (주의 필요)
# |z| > 3: 상위/하위 0.3% (이상치)
```

# 불확실성 정량화

## 신뢰구간 (Confidence Interval)

```python
import numpy as np
from scipy import stats

data = np.random.normal(1200, 50, 100)

# 95% 신뢰구간 계산
mean = np.mean(data)
sem = stats.sem(data)  # 표준오차
ci = stats.t.interval(0.95, len(data)-1, loc=mean, scale=sem)

print(f"평균: {mean:.1f}")
print(f"95% 신뢰구간: [{ci[0]:.1f}, {ci[1]:.1f}]")
```
