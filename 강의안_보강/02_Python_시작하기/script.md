## Python 기본 문법 (5분)

### 5가지 자료형 [2분]

> 이제 Python 문법을 배워봅시다.
>
> Python에서 꼭 알아야 할 자료형은 **5가지**입니다.
>
> **1. 정수(int)**: 소수점 없는 숫자
> 생산량 1000개, 라인 번호 3번 같은 거요.
>
> **2. 실수(float)**: 소수점 있는 숫자
> 불량률 0.025, 온도 85.7도 같은 거요.
>
> **3. 문자열(str)**: 텍스트
> "센서 모듈 A", "정상" 같은 거요.
>
> **4. 리스트(list)**: 여러 데이터를 순서대로 담는 자료형
> 대괄호로 만들어요. [82, 85, 88, 84]
>
> **5. 딕셔너리(dict)**: 이름표를 붙여 데이터를 저장
> 중괄호로 만들어요. {"온도": 85, "습도": 45}
>
> 이 5가지가 데이터의 기본 단위입니다.

### f-string 사용법 [1분]

> 문자열에서 가장 많이 쓰는 기능이 **f-string**입니다.
>
> 문자열 앞에 f를 붙이고, 중괄호 안에 변수를 넣으면
> 그 값이 문자열에 포함돼요.
>
> ```python
> temp = 85.7
> print(f"온도: {temp}도")
> # 결과: 온도: 85.7도
> ```
>
> 숫자 포맷도 할 수 있어요.
>
> ```python
> rate = 0.025
> print(f"불량률: {rate:.1%}")
> # 결과: 불량률: 2.5%
> ```
>
> .1%가 소수점 1자리 퍼센트 표시라는 뜻이에요.

### 조건문 if [1min]

> **조건문 if**는 "만약 ~라면 ~해라"를 코드로 표현합니다.
>
> ```python
> if temperature > 90:
>     print("경고")
> elif temperature > 80:
>     print("주의")
> else:
>     print("정상")
> ```
>
> 핵심 문법 두 가지!
>
> 첫째, 조건 뒤에 **콜론(:)**을 꼭 붙여야 합니다.
> 둘째, 다음 줄은 **들여쓰기**(스페이스 4칸)를 해야 해요.
>
> 들여쓰기를 안 하면 에러가 납니다.
> Python은 들여쓰기로 코드 블록을 구분하거든요.

### 반복문 for [1min]

> **반복문 for**는 같은 작업을 여러 번 실행할 때 사용합니다.
>
> ```python
> temperatures = [82, 85, 88, 95, 84]
>
> for temp in temperatures:
>     print(f"온도: {temp}도")
> ```
>
> 리스트의 각 요소가 차례로 temp 변수에 들어가면서 반복됩니다.
>
> 인덱스도 같이 필요하면 **enumerate**를 사용해요.
>
> ```python
> for i, temp in enumerate(temperatures):
>     print(f"{i+1}번째: {temp}도")
> ```
>
> i는 0부터 시작하는 인덱스, temp는 값입니다.

>
> 불량률에 따라 품질 등급을 판정해봅시다.

> ```python
> defect_rate = 0.025  # 불량률 2.5%
>
> if defect_rate <= 0.01:
>     grade, msg = "A", "우수"
> elif defect_rate <= 0.03:
>     grade, msg = "B", "양호"

>     grade, msg = "D", "개선 필요"
>
> print(f"품질 등급: {grade} ({msg})")
> ```
>
> 불량률 2.5%는 3% 이하이므로 B등급이 나옵니다.
>
> defect_rate 값을 0.005로 바꿔보세요. 무슨 등급이 나오나요?
> (A등급!)

### 실습 3: 반복문으로 온도 분석 [3min]

> 세 번째 실습입니다.
>
> 시간별 온도 데이터를 분석해봅시다.
>
> ```python
> temperatures = [82, 85, 88, 95, 92, 89, 91, 86, 84]
> threshold = 90  # 경고 기준
>
> print("=== 시간별 온도 모니터링 ===")
> warnings = []
>
> for i, temp in enumerate(temperatures):
>     hour = 9 + i  # 9시부터 시작
>     if temp > threshold:
>         print(f"[경고] {hour}시: {temp}도")
>         warnings.append(f"{hour}시")
>     else:
>         print(f"[정상] {hour}시: {temp}도")
>
> print(f"\n경고 시간대: {', '.join(warnings)}")
> ```
>
> 90도를 초과한 시간대에 경고가 출력됩니다.
> 12시, 13시, 15시에 경고가 나왔네요.

### 실습 4: 딕셔너리로 설비 관리 [3min]

> 네 번째 실습입니다.
>
> 딕셔너리로 여러 설비의 상태를 관리해봅시다.
>
> ```python
> equipment = {
>     "EQ-001": {"온도": 82, "진동": 0.3, "상태": "정상"},
>     "EQ-002": {"온도": 91, "진동": 0.8, "상태": "주의"},
>     "EQ-003": {"온도": 85, "진동": 0.4, "상태": "정상"},
>     "EQ-004": {"온도": 96, "진동": 1.2, "상태": "경고"}
> }
>
> print("=== 설비 상태 점검 ===")
> for eq_id, data in equipment.items():
>     print(f"{eq_id}: 온도 {data['온도']}도, 상태 {data['상태']}")
>
> # 이상 설비 필터링
> abnormal = [eq for eq, data in equipment.items() if data["상태"] != "정상"]
> print(f"\n이상 설비: {', '.join(abnormal)}")
> ```
>
> 딕셔너리 안에 딕셔너리를 넣을 수 있어요.
> 이런 구조를 **중첩 딕셔너리**라고 합니다.
