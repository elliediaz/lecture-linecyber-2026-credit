## 커리큘럼 구조

전체 AI 개발 파이프라인을 4개 파트로 구성:

1. **기초 (1-4차시)**: 환경 설정, Python 기초, 공공데이터, NumPy/Pandas
2. **데이터 분석 및 통계 (5-19차시)**: 기술통계, 확률, 가설검정, 전처리, 회귀분석, PCA
3. **AI 기초 (20-36차시)**: ML/DL 개념, 신경망, TensorFlow/Keras, CNN, YOLO, RNN/LSTM, GAN, Transformer, LLM
4. **배포 (37-40차시)**: Streamlit, FastAPI, 모델 배포, 종합 프로젝트
